# CampusWorld — Plugin

Plugin Paper/Spigot do **CampusWorld** — ponte entre o servidor Minecraft e o backend.

## Responsabilidades

- Whitelist e convites (`/invite`)
- Sincronização de jogadores com a API
- Probation e restrições in-game
- Claims e proteção de terreno
- Emissão de eventos de auditoria

## Stack (planejado)

- Java 21
- Paper API 1.21+

## Build

```bash
# Em breve: ./gradlew build
# Artefato: build/libs/CampusWorld-*.jar → plugins/
```
